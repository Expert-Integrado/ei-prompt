---
description: Atualiza os agentes EiPrompt na pasta atual via npx (@expertzinhointegrado/ei-prompt@latest)
---

Execute o comando de atualização do pacote EiPrompt:

```bash
npx @expertzinhointegrado/ei-prompt@latest
```

Depois confirme ao usuário que a atualização foi executada e, se útil, liste brevemente o que mudou em `modelo/` (ex: `ls modelo/`).
